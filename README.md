# AtomQuest

A real-time video support platform built for customer support teams using Node.js, WebSockets, and vanilla JavaScript.

## Live Demo

- **Live App:** [https://atomquest-4sim.onrender.com](https://atomquest-4sim.onrender.com)
- **Admin Dashboard:** [https://atomquest-4sim.onrender.com/?admin=1](https://atomquest-4sim.onrender.com/?admin=1)

## Features

- Role-based access for Agent and Customer
- Real-time video and audio support
- Session invite links
- Live chat
- Camera and microphone controls
- File sharing
- Server-side call recording
- Reconnect handling
- Admin dashboard

## Tech Stack

- Node.js
- Native WebSocket
- HTML
- CSS
- Vanilla JavaScript
- MediaRecorder API
- JSON/local file storage

## Setup

```bash
git clone https://github.com/Praveenpadidapu/AtomQuest.git
cd AtomQuest
npm install
npm run dev
```

Open in browser:

```text
http://localhost:3000
```
